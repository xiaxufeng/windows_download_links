## 校验文件完整性

下载完成后，建议核对 **SHA1 / SHA256 / MD5** 校验值，确认镜像与官方发布版本一致。

**Windows PowerShell：**

```powershell
Get-FileHash .\xxx.iso -Algorithm SHA256
Get-FileHash .\xxx.iso -Algorithm SHA1
Get-FileHash .\xxx.iso -Algorithm MD5
```

**macOS / Linux：**

```bash
shasum -a 256 xxx.iso
shasum -a 1   xxx.iso
md5sum        xxx.iso
```
